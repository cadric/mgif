Install Fedora Gnome

An easy script to Install a Minimal Version of Fedora and Gnome. Install a small curated selection of flatpak apps to get started with Decide if you want a pure Gnome experince, get a windows style look or macOS look for your desktop. Download and install Fedora Everything from: https://fedoraproject.org/everything/download -Create a user with password and where you want fedora installed. -Do not select additional software. -Reboot into cli Login with your user and type: sudo su - Then type: bash <(curl -fsSL ifg.sh) Answer the questions the script have for you and let it do its magic! Reboot into your new Minimal Fedora Gnome desktop
