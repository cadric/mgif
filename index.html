Minimal Fedora GNOME Setup

An easy script to install a minimal version of Fedora with GNOME desktop environment. Install a small curated selection of flatpak apps to get started with and decide if you want a pure GNOME experience, Windows-style look, or macOS appearance for your desktop.

Installation process demonstration

01

Download Fedora Everything

Download and install Fedora Everything from the official website:

https://fedoraproject.org/everything/download
  • Create a user with password and select where you want Fedora installed
  • Do not select additional software during installation
  • Reboot into CLI (command line interface)
02

Login and Switch to Root

Login with your user account and switch to root privileges:

sudo su -
03

Run the Installation Script

For security, verify the script integrity before execution:

🔐 Security Verification (Recommended)

Verify the script's SHA512 hash before running:

curl -fsSL ifg.sh | sha512sum
Expected SHA512: 41b3e538f1d4f8ffcffa2505b81271575dca517310956846eae9c787b2cffcf78de4dce00ff5c51f9de29b1ad7358bc64e6a80af72a4c5be68b62f83cb4353b8

Alternative verification: View hash on Pastebin

⚠️ Only proceed if the hashes match exactly!

🚨 Security Warning

Never run unreviewed scripts from the internet! This script modifies your system. Always verify the source code before execution.

Review the source code on GitHub before proceeding.

Run the automated installation script:

bash <(curl -fsSL ifg.sh)

The script will ask you configuration questions to customize your installation.

04

Answer Configuration Questions

The script will prompt you for your preferences:

  • Installation scope: System-wide or user-specific apps
  • App bundle: Install curated applications
  • Wallpapers: Install beautiful desktop backgrounds
  • Desktop style: Pure GNOME, Windows-like, or macOS-style
  • GRUB menu: Hide or show boot menu
05

Enjoy Your New Desktop

Let the script work its magic and then reboot into your new minimal Fedora GNOME desktop!
